== Description ==

= Plugin myposeo pour WordPress =

[myposeo](http://www.myposeo.com) est un outil simple, efficace et complet qui vous permet de suivre votre positionnement jour après jour sur un grand nombre de moteurs de recherche.

Le plugin myposeo dashboard pour WordPress vous permet d'afficher, dans le back-office de votre site ou de votre blog WordPress les résultats de positionnement de votre compte myposeo.

La dernière version 1.1.3 du pugin a été testée avec succès avec la dernière version de WordPress.

== Installation ==

Vous pouvez installer facilement le plugin myposeo dashboard depuis l'installation automatique de votre back-office. Cette installation est recommandée, elle est rapide et vous permettra de mettre facilement votre plugin à jour.

Vous pouvez aussi faire l'installation manuellement :

1. Téléchargez le plugin myposeo et copiez le répertoire `myposeo-dashboard` dans le dossier `wp-content/plugins/` de votre site WordPress.
2. Activez le plugin depuis le menu 'Extension.
3. Allez sur la page de configuration afin d'inscrire/récupérer votre clé API myposeo.

== Frequently Asked Questions ==

= Qu'est ce que myposeo ? =

[myposeo](http://www.myposeo.com) est un outil simple, efficace et complet qui vous permet de suivre votre positionnement jour après jour sur un grand nombre de moteurs de recherche.

= Quelles sont les offres myposeo ? =

Retrouvez toutes les offres myposeo sur la page "[Nos offres](https://www.myposeo.com/fr/packs)".

= Comment puis je contacter le service myposeo.com ? =

Vous pouvez contacter myposeo via me formulaire sur [la page de contact](https://www.myposeo.com/fr/contact).

= Comment s'inscrire sur myposeo.com ? =

Vous pouvez vous inscrire sur [myposeo](www.myposeo.com) directement depuis le [formulaire d'inscription personnalisé](https://www.myposeo.com/fr/register).

= Où puis-je trouver ma clé API ? =

Vous pouvez trouver votre clé API sur la page "[Configuration](https://www.myposeo.com/fr/account/configuration)" de votre compte myposeo.

= Le plugin est-il gratuit ? =

Oui le plugin est entièsrement gratuit.

= Le plugin peut-il être utilisé par les administrateurs/utilisateurs de mon compte ? =

Oui, tous les utilisateurs et administrateurs peuvent utiliser le plugin myposeo.

= Avec quelle version de WordPress le plugin est-il compatbile ? =

Le plugin est offciellement compatible avec la version WordPress 3.0.1 et supérieur.

= Je n'arrive pas à installer le plugin sur mon site, que faire ? =

N'hésitez pas à nous contacter via [le formulaire de contact](https://www.myposeo.com/fr/contact) ou depuis [le support](https://www.myposeo.com/fr/account/support) de votre compte myposeo.

= J'ai oublié ma clé d'API myposeo, que faire ? =

Vous pouvez vous connecter avec vos identifiants myposeo sur la page de configuration du plugin pour récupérer votre clé API myposeo.

= Puis-je obtenir une clé API gratuitement ? =

L'API est disponible pour tous les utilisateurs gratuitement, vous pouvez trouver votre clé API sur la page "[Configuration](https://www.myposeo.com/fr/account/configuration)" de votre compte myposeo.
